# 📘 처음부터 끝까지 따라하는 설치 가이드 (비개발자용)

목표: **컴퓨터 없이도 매일 아침 7시에 글 10개가 자동 생성되고, 아이폰 앱으로 확인·게시**까지.
예상 소요: 약 30~40분 (한 번만 설정하면 끝).

준비물: ① 구글 계정 ② LLM API 키(OpenAI 등) ③ (자동 게시 시) 워드프레스 사이트.
순서대로만 따라오면 됩니다. 모르는 단어가 나와도 그대로 복사·클릭하면 동작합니다.

---

## STEP 0. LLM API 키 발급 (글을 쓰는 엔진)

셋 중 하나만 있으면 됩니다. 가장 쉬운 **OpenAI** 기준:

1. https://platform.openai.com 접속 → 구글 계정으로 로그인.
2. 우측 상단 프로필 → **Billing**에서 카드 등록 후 소액($5~10) 충전.
   - `gpt-4o-mini`는 글 1개당 약 1~3원 수준이라 하루 10개라도 매우 저렴합니다.
3. 좌측 **API keys → Create new secret key** → 키를 복사해 메모장에 임시 보관.
   - 키는 `sk-...` 로 시작하며 **한 번만 보여줍니다**. 꼭 복사해 두세요.

> 무료로 시작하고 싶으면 **Google Gemini**(`gemini-1.5-flash`)가 무료 한도가 넉넉합니다.
> https://aistudio.google.com → Get API key. 이 경우 STEP 3에서 provider를 `gemini`로 설정.

---

## STEP 1. GitHub 가입 & 저장소 만들기

1. https://github.com → **Sign up** (구글 이메일로 가입, 무료).
2. 로그인 후 우측 상단 **+** → **New repository**.
3. 설정:
   - **Repository name**: `blog-auto` (아무 이름이나 가능, 영문/숫자)
   - **Public** 선택 (무료 Pages를 쓰려면 Public 권장. 비밀 키는 코드에 안 들어가니 안전)
   - 나머지는 기본값 → **Create repository**.

---

## STEP 2. 프로젝트 파일 업로드 (git 몰라도 됨)

웹에서 드래그&드롭으로 올립니다.

1. 방금 만든 저장소 페이지에서 **uploading an existing file** 링크 클릭
   (또는 **Add file → Upload files**).
2. 내 컴퓨터의 **`adsense-blog-automation` 폴더 안의 모든 파일/폴더**를 통째로 드래그.
   - `.github` 폴더(워크플로)도 반드시 함께 올라가야 합니다. 폴더째 드래그하면 포함됩니다.
3. 하단 **Commit changes** 클릭.

> ⚠️ 업로드 시 `config.json`은 올라가도 괜찮습니다(키 자리는 자리표시자). 실제 키는 STEP 3의
> **시크릿**으로 넣으므로 코드에 노출되지 않습니다.

---

## STEP 3. 시크릿(비밀 키) 등록

저장소에서 **Settings → 좌측 Secrets and variables → Actions → New repository secret**.
아래 항목을 **하나씩** 추가합니다 (Name = 이름, Secret = 값).

필수:

| Name | Secret(값) |
|---|---|
| `LLM_PROVIDER` | `openai` (제미니면 `gemini`) |
| `LLM_API_KEY` | STEP 0에서 복사한 키 (`sk-...`) |
| `LLM_MODEL` | `gpt-4o-mini` (제미니면 `gemini-1.5-flash`) |
| `LANGS` | `ko,en` (한국어만 원하면 `ko`) |

자동 게시까지 원하면 추가 (나중에 해도 됨):

| Name | Secret(값) |
|---|---|
| `WP_ENABLED` | `true` |
| `WP_SITE` | `https://내블로그.com` |
| `WP_USER` | 워드프레스 아이디 |
| `WP_APP_PASSWORD` | 응용 프로그램 비밀번호 (STEP 6 참고) |
| `WP_STATUS` | `draft` (검토 후 발행) 또는 `publish` |

> `WP_ENABLED`를 빼거나 `false`로 두면, 자동 게시는 안 하고 글만 생성됩니다(앱에서 복붙/게시).

---

## STEP 4. 자동 실행 켜고 한 번 테스트

1. 저장소 상단 **Actions** 탭 클릭 → 워크플로 사용 동의 버튼이 보이면 **I understand... enable**.
2. 좌측 **매일 블로그 자동 생성** 선택 → 우측 **Run workflow → Run workflow** (지금 즉시 테스트).
3. 1~3분 후 초록 체크 ✓ 가 뜨면 성공. 빨간 ✗ 면 로그를 눌러 어떤 시크릿이 비었는지 확인.
4. 이후로는 **매일 한국시간 오전 7시**에 자동 실행됩니다(설정 불필요).

> 실행이 끝나면 저장소의 `dashboard/data/latest.json`이 오늘 글로 갱신됩니다.

---

## STEP 5. 대시보드를 인터넷에 공개 (GitHub Pages)

1. **Settings → 좌측 Pages**.
2. **Source**: `Deploy from a branch` → **Branch**: `main` / 폴더 `/ (root)` → **Save**.
3. 1~2분 뒤 상단에 주소가 생깁니다:
   `https://<내아이디>.github.io/blog-auto/dashboard/`
4. 이 주소를 PC 브라우저에서 먼저 열어 글이 보이는지 확인.
   - 데이터 파일 주소(메모): `https://<내아이디>.github.io/blog-auto/dashboard/data/latest.json`

---

## STEP 6. (선택) 워드프레스 게시 준비

자동 게시 또는 폰에서 게시하려면 **응용 프로그램 비밀번호**가 필요합니다.

1. 워드프레스 관리자 → **사용자 → 프로필**.
2. 맨 아래 **응용 프로그램 비밀번호** → 이름 입력(예: `auto-blog`) → **새 응용 프로그램 비밀번호 추가**.
3. 생성된 `abcd efgh ijkl ...`(공백 포함) 전체를 복사.
   - 자동 게시용이면 STEP 3의 `WP_APP_PASSWORD`에 입력.
   - 폰에서 직접 게시할 거면 STEP 8의 앱 설정에 입력.

---

## STEP 7. 아이폰에 앱 설치

1. 아이폰 **사파리(Safari)**로 STEP 5의 `.../dashboard/` 주소 접속.
   (크롬 말고 사파리여야 설치가 됩니다.)
2. 하단 가운데 **공유 버튼**(□ 위에 화살표) 탭.
3. 목록을 내려 **홈 화면에 추가** → 우측 상단 **추가**.
4. 홈 화면에 생긴 🚀 아이콘으로 열면 **전체화면 앱**으로 실행됩니다.

---

## STEP 8. 앱에서 설정

앱 우측 상단 **⚙️** 탭:

1. **데이터 주소**: STEP 5에서 메모한 `.../data/latest.json` 주소 붙여넣기.
   → 이제 매일 7시 생성된 새 글이 앱에 자동으로 들어옵니다.
2. (폰에서 직접 게시할 경우) **워드프레스** 사이트·아이디·앱비밀번호 입력, 게시 방식은 처음엔 `임시저장`.
3. **저장**.

이제 글 카드에서:
- **📤 게시**: 폰에서 바로 워드프레스로 발행
- **📋 HTML / 텍스트**: 복사해서 어디든 붙여넣기
- **미리보기**: 글 내용 확인

## 자주 막히는 부분

| 증상 | 원인 / 해결 |
|---|---|
| Actions가 빨간 ✗ | 로그 클릭 → 보통 `LLM_API_KEY` 미입력/오타. 시크릿 다시 확인 |
| Pages 주소 404 | 저장 후 2분 대기. Branch가 `main`/`root`인지 확인 |
| 앱에 글이 안 보임 | ⚙️ 데이터 주소가 `latest.json`까지 정확한지, 한 번 ↻ 새로고침 |
| 폰 게시 시 실패 | CORS 문제. README 9-4의 워드프레스 코드 추가, 또는 자동 게시(STEP 3)로 전환 |
| 비용 걱정 | `gpt-4o-mini`는 하루 10글 기준 월 몇 백 원대. 무료는 Gemini 사용 |

## 매일 일어나는 일 (설정 후)

```
오전 7:00  GitHub Actions 자동 실행
   → 핫이슈 5 + 세분화 5 = 한/영 글 생성
   → latest.json 갱신 (자동)
언제든   아이폰 앱 열기 → 오늘 글 확인 → 게시 or 복붙
```

설정은 한 번만. 이후에는 **앱만 열면 됩니다.**

## ⚠️ AdSense 운영 주의

구글은 무검수 자동 대량 발행을 저품질로 봅니다. 안전하게:
- 처음엔 `draft`(임시저장)로 두고 **사람이 검토 후 발행**
- 수치·링크는 한 번 확인 (AI가 틀릴 수 있음)
- 세분화(롱테일) 글이 경쟁이 약해 노출·수익에 유리
